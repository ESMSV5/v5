Rebuilding a Legacy Football Simulation Engine from First Principles

This project rebuilds the ESMS match engine from the ground up using modern C++ structure and deterministic logic. The goal is not to copy the original code, but to recreate its behaviour with clarity, control, and extensibility.

The engine simulates a full 90-minute match using a seeded RNG. Every outcome is reproducible. Every step in the match flow is explicit and testable.
